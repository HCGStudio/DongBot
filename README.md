# Dong! Bot

Dong! Bot是一个多后端，支持插件，一部驱动的dotnet平台上的聊天机器人。

## 后端支持

目前支持或者准备支持以下框架或者软件

- [x] CqHttp (通过[cqhttp.Cyan](https://github.com/frank-bots/cqhttp.Cyan))
- [ ] mirai (未在近期计划中，你可以暂且使用[cqhttp-mirai](https://github.com/yyuueexxiinngg/cqhttp-mirai)在mirai上进行替代)
- [ ] 其他聊天软件支持
